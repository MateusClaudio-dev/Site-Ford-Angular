O .cdr (ChangeDetectorRef) é uma ferramenta do Angular usada para gerenciar manualmente a atualização da tela.
Uso do this.cdr.detectChanges() => Notifica o Angular que novos dados chegaram (geralmente dentro de funções assíncronas, como um .subscribe()) e força o framework a atualizar o HTML imediatamente com os novos valores, evitando que a tela fique desatualizada mesmo com os dados já carregados no TypeScript.
